# Sora Video Editor

Full-stack AI video generation platform built with:

- Turborepo + pnpm
- Next.js (App Router)
- Express API
- PostgreSQL + Prisma
- Redis + BullMQ
- Cloudinary (media storage)
- Worker (video generation jobs)
- NextAuth (OAuth + Credentials)
- Docker + docker-compose

## Quick Start

### 1. Requirements
- Node.js 20+
- pnpm
- Docker & Docker Compose
- Google OAuth app (optional for login)
- Cloudinary account
